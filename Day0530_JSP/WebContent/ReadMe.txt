쿠키 Cookie
HTTP 통신은 비연결 통신 : 한번 요청하고 나서 응답을 하면 연결이 끊김
지속적인 데이터가 필요할 때, 상태를 저장하고 사용하기 위함
응답을 할 때 쿠키를 만들어서 전송 : 브라우저가 응답의 쿠키를 저장하고 있다가
요청할 때 쿠키를 요청과 함께 전송

해당 웹브라우저가 웹서버에 요청할 때 쿠키를 같이 보냄으로 써 웹서버는 해당브라우저가 어떤정보를
가지고 있었는지 알 수 있다.

쿠키가 session의 정보를 가지고 있음
쿠키는 생성은 서버가 하고, 보관은 브라우저가 함
쿠키는 Cookie 클래스를 이용 : java코드를 이용해서 생성
