JSP 기본 객체
JSP에서 바로 사용할 수 있는 JSP에서 제공하는 객체
	request : 클라이언터의 요청정보를 저장하는 객체
	response : 응답을 위한 정보를 저장하는 객체
	pageContext : 하나의 jsp페이지 정보를 저장 하는 객체
	session : http session 정보를 저장하는 하는객체
	application : 웹어플리케이션 정보를 저장하는 객체
	out : jsp페이지가 결과를 출력할 때 사용하는 출력 스트림
	config : 설정정보 저장 객체
	exception : 예외 정보 저장 객체, 에러페이지에서만 사용가능

	웹 프로젝트는 크게 네 가지 영역으로 구분 가능
	page영역
	request영역
	session영역
	application 영역
