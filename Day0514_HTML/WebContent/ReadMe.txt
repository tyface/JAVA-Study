html 요소는 두가지로 분류
block 요소: 화면의 가로영역을 모두 차지하는 요소
in-line 요소: 내용만큼만 영역을 차지하는 요소, 옆으로 다른요소가 올 수 있음

<h>: 문서내에서 제목을 표시할 때 사용, h1~h6까지 중요순으로 사용
<p>: 문서에서 문장의 단락을 만드는 역할
<b>,<strong>: 텍스트 bold 처리
<mark>: 텍스트 강조(텍스트칸 백크라운드 컬러)
<blockquote>: 인용문 작성할때 사용
<hr>: 한 줄 개행 하고나서 줄긋기 태그
<br>: 한 줄 개행 태그
<i>,<em>: 텍스트 기울기
<span>: 요소내에서 특정 영역을 지정할 때 사용
<pre>: 특수문자를 그대로 사용하고 싶은때
<ol>: 숫자가 있는 리스트
<ul>: 숫자가 없는 리스트
<li>: 목록의 내용을 표시하는 태그
<anchor> 웹 문서간에 이동이나 , 문서가 너무 길때 문서내 이동

id속성: html 문서 내에서 요소들을 구분하는 구분자 역할을 함
문서내에서 유일한 값을 가져야 한다. (문서 뿐만아니라 프로젝트에 있는 모든 문서내에서 유일한 값을 가지면 좋다)

target 속성: 링크시 화면 표시방법을 설정
_blank: 새탭 또는 새창에서 열기
_self: 기본값, 현제 화면에서 링크가 열림
_parent: 프레임을 사용했을 경우에 내용을 부모 프레임에 표시
