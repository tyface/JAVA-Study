Html (Hypertext Markup Language)
웹브라우저에서 보이는 화면을 그리기 위한 언어
Hypertext: 링크 를 통해 다른 문서나 사이트로 이동할 수 있는 기능

Tag(태그)로 구성
<태그이름>내용</태그이름>
태그를 이용해서 문서를 구분하기 때문에 markup이라고 한다.
(이미지,버튼,글자 ,제목 인지 ....)

태그에는 속성값(class, id, style 등등...)
<tag 속성이름1 = "속성값" 속성이름2 = "값" ></tag>
ex)<a href = "www.naver.com" class = "myclass"></a>

많은 곳에서 이전 HTML과 구분하기 위해서 HTML5 라고 사용
